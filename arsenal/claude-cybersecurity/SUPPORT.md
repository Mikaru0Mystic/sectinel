## Before Opening an Issue

1. Check the [README](README.md) for usage instructions
2. Search [existing issues](https://github.com/AgriciDaniel/claude-cybersecurity/issues) for duplicates
3. Ensure you're using the latest version (`git pull` in skill directory)

## False Positive Reports

If the skill flags something that isn't a real vulnerability:
- Include the code snippet that was flagged
- Explain why it's a false positive (e.g., framework protection, compensating control)
- This helps us improve the false-positive suppression rules
